 🛒 SmartCart - AI Shopping Assistant

Overview :

SmartCart is an AI-powered shopping assistant built using Streamlit.
It allows users to explore gadgets, manage a shopping cart, and get intelligent product recommendations using AI.

---

✨ Features :

* 🔐 User Login System
* 🛍️ Browse products (Laptops, Smartphones, Smartwatches)
* ➕ Add items to cart
* ❌ Remove items from cart
* 🧹 Clear cart functionality
* 💰 Automatic total price calculation
* 🤖 AI-powered assistant for product suggestions & comparisons

---

🧠 How It Works

* Users log in to access the app
* Products are displayed with price and description
* Users can add/remove items from the cart
* AI assistant helps in selecting products based on user queries

---

🛠️ Tech Stack

* Python
* Streamlit
* Google Generative AI (Gemini API)

---

▶️ How to Run Locally

 1. Clone the repository

```bash
git clone 05maherashaikh/SmartCart 
cd smartcart
```

 2. Install dependencies

```bash
pip install -r requirements.txt
```

 3. Run the app

```bash
streamlit run main.py
```

---

 🔐 Environment Variables (IMPORTANT)

Create a `.streamlit/secrets.toml` file:

```toml
GEMINI_API_KEY = "your_api_key_here"
```
